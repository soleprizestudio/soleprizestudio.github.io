# SolePrize Studio

SolePrize Studio's brand site — built with [Astro](https://astro.build) on the
[AstroWind](https://github.com/arthelokyo/astrowind) template (MIT license,
see `LICENSE.md`).

## Local development

```bash
npm install
npm run dev
```

## Deployment

Deploys automatically to GitHub Pages via GitHub Actions on every push to
`main` (see `.github/workflows/deploy.yml`).
