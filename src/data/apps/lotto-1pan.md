이번 주 로또 1등은 어디에서 나왔을까요?

로또 1판은 매주 발표되는 로또 1등 당첨 판매점 정보를 쉽고 빠르게 확인할 수 있는 앱입니다.

## 주요 기능

- 이번 주 로또 1등 당첨 판매점 확인
- 전국 당첨 판매점 위치 검색
- 지도에서 판매점 위치 확인
- 내 주변 로또 판매점 찾기

로또를 구매하기 전, 이번 주 행운이 시작된 판매점을 로또 1판에서 확인해보세요.

※ 본 앱은 로또 당첨을 예측하거나 보장하지 않습니다. 공개된 로또 당첨 판매점 정보를 제공하는 서비스입니다.
